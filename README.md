# HR Analytics Dashboard (Interactive Dashboard using Power BI)
## Project Objective
The objective of this project is to analyze employee data and understand the key factors that influence employee attrition within the organization. This dashboard helps HR teams identify patterns in employee turnover and make data-driven decisions to improve employee retention.

## Dataset Used
HR Analytics Dataset
- <a href="https://github.com/ragumass55/Data-Analysis-Dashboard/blob/main/HR%20analytcs.project.pbix">Dataset</a>

## Questions (KPIs)
* What is the overall employee attrition rate?
* Which department has the highest attrition?
* What is the employee distribution by age group?
* How does attrition vary by gender?
* Which job roles have the highest attrition?
* How many employees are currently active vs left the company?
* What is the attrition trend based on years at company?

## Process
* Imported the HR dataset into Power BI.
* Verified the dataset for missing values and data inconsistencies.
* Cleaned and transformed the dataset using Power BI Power Query.
* Created calculated measures and KPIs for HR analysis.
* Designed visualizations including charts, cards, and graphs.
* Combined all visuals into an interactive dashboard with filters and slicers.

## Project Insights
* Certain departments show higher employee attrition compared to others.
* Employees with fewer years at the company tend to leave more frequently.
* Attrition varies across different job roles and salary levels.
* Workforce demographics reveal patterns in employee retention.

## Final Conclusion
The HR Analytics Dashboard provides valuable insights into employee attrition patterns and workforce distribution. By analyzing these trends, organizations can identify potential problem areas and implement strategies to improve employee satisfaction and retention.
This project demonstrates how Power BI can be used to transform HR data into meaningful business insights through interactive dashboards.
